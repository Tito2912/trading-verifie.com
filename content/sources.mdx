Pour éviter les erreurs, on privilégie des **sources primaires** : pages de prix/frais, conditions, centres d’aide officiels, et documents de risque.

## Types de sources (priorité)

1. **Pages de frais/pricing** officielles (tarifs, spreads, conversions, retraits).
2. **Conditions / risk disclosures** (ce que vous achetez vraiment, avertissements CFD/crypto).
3. **Help center** (procédure KYC, dépôts/retraits, limites).
4. **Pages de sécurité** (2FA, protections, procédures de récupération).
5. **Registres / autorités** quand c’est applicable (statut, entité, avertissements).

## Méthode de vérification “2 minutes”

Avant d’agir (dépôt, trade, copie, retrait), vérifiez :

- la **date** du document (et si une version locale existe pour votre pays)
- si la page parle de l’**actif** ou d’un **CFD**
- le **coût total** (frais explicites + spread + conversion)
- les **conditions** de retrait (délai, frais, justificatifs)

## Pourquoi on insiste

Les plateformes évoluent : ce qui est vrai dans un pays / une devise / un type de compte peut être faux ailleurs.  
Un “bon guide” doit donc vous apprendre à **vérifier**, pas seulement à lire.

## Avertissement

Le contenu est informatif et ne constitue pas un conseil financier. Les produits à effet de levier (CFD) et la crypto impliquent un risque élevé.
